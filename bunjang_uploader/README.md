# ⚡ 번개장터 일괄 업로드 프로그램

## 📋 요구사항

- Python 3.10 이상
- Google Chrome 브라우저
- ChromeDriver (자동 설치)

## 🚀 설치 방법

```bash
# 1. 패키지 설치
pip install selenium webdriver-manager

# 2. 실행
python main.py
```

## 📄 CSV 파일 형식

| 컬럼 | 설명 | 예시 |
|------|------|------|
| 이미지경로 | 이미지 절대경로 (여러 장은 `|` 구분자) | `C:/img/a.jpg\|C:/img/b.jpg` |
| 상품명 | 상품 제목 | `아이폰 14 128GB` |
| 카테고리 | 번개장터 카테고리명 | `디지털/가전` |
| 상품상태 | 새상품 / 거의새것 / 중고 | `중고` |
| 설명 | 상품 설명 | `6개월 사용, 케이스 포함` |
| 가격 | 숫자만 (원 단위) | `50000` |
| 수량 | 수량 | `1` |

## ⚙️ 사용 방법

1. **로그인 설정 탭**: 네이버 아이디/비밀번호 입력 후 `브라우저 실행 & 로그인`
2. **상품 목록 탭**: CSV 파일 불러오기 후 `업로드 시작`
3. **실행 로그 탭**: 진행 상황 모니터링

## ⚠️ 주의사항

- ChromeDriver는 Chrome 버전과 일치해야 합니다.
- 2단계 인증 사용 시 자동으로 팝업이 열리면 직접 인증하세요.
- 과도한 자동화는 계정 제한의 원인이 될 수 있으니 개인 사용 목적으로만 사용하세요.
- 이미지 파일은 로컬 경로(절대 경로)로 지정해야 합니다.
